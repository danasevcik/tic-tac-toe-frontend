# Tic Tac Toe

This application offers users:
* Login and signup features
* 3 levels of difficulty
* Ability to view high scores
* Ability to customize the color of the scoreboard

## Technologies Used
* Ruby on Rails
* React
* JWT Authorization

## How to Use
1. Run the rails server
```
rails s
```
2. Start React
```
npm start
```
